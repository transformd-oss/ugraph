parse

  • parse(options: { data: unknown; onConflict?: "abort" | "merge" | "ignore"; types?: unknown; onNode?: any; onTyped?: any }): Result<Result.Ok<Graph>, Result.Err<"DATA_INVALID", { errors: DataError[] }> | Result.Err<"TYPES_INVALID", { errors: TypesError[] }>>

  • Given data, will return a runtime Graph with all Nodes recursively resolved and mapped in .nodes for access via. that Node's $id. Accepts any data structure, although Objects and/or Arrays are most useful.

    If given types, will also validate all Objects with a $type property defined. Accepts a Map of Nodes and/or Validators OR a data-like definition which will be automatically parsed for it's Nodes. Type-definition Nodes use it's $id property to determine which $type name to validate on a Object.

    Parameters

    • options: { data: unknown; onConflict?: "abort" | "merge" | "ignore"; types?: unknown; onNode?: any; onTyped?: any }
      • data: unknown

        Data to walk and extract nodes from.

      • Optional onConflict?: "abort" | "merge" | "ignore"

        Default: "abort"

        • "abort", return error on conflicting node definition.
        • "merge", merge props of all conflicting node definitions (slower).
        • "ignore", skips conflicts (only first instance is used).
      • Optional types?: unknown

        Type definitions to enforce node schemas via $type declarations.

        • if given Map<NodeID, TypedNode> will create validators automatically.
        • if given Map<TypeName, Validator> will use as is.
        • otherwise, will be parsed via resolve automatically for validators.
      • onNode?:function
        • onNode(id: string, node: Node<Obj>): void

        • Optional callback fired once per-every complete Node object, with $id.

          Parameters

          • id: string
          • node: Node<Obj>

          Returns void

      • onTyped?:function
        • onTyped(type: string, typed: Typed<string, Obj>): void

        • Optional callback fired once per-every complete Typed object, with $type.

          Parameters

          • type: string
          • typed: Typed<string, Obj>

          Returns void

    Returns Result<Result.Ok<Graph>, Result.Err<"DATA_INVALID", { errors: DataError[] }> | Result.Err<"TYPES_INVALID", { errors: TypesError[] }>>
